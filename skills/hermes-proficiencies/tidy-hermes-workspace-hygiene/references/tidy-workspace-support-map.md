# Tidy Workspace Support File Map

Load support files by mode; do not load everything by default.

- `references/modes/normal-workspace-hygiene.md` — ordinary git/workspace edits, file routing, naming, origin/changelog, and shared play sequence.
- `references/modes/protected-wiki.md` — protected iCloud/Citadel/wiki paths; read-only default and exact write-approval requirements.
- `references/modes/skill-source-runtime-hygiene.md` — Hermes skill source/runtime sync, private proficiencies vs soulbound/tool skills, category placement, and private repo rules.
- `references/modes/destructive-operations.md` — deletes, overwrites, bulk moves, recursive commands, broad dangerous approval pitfalls.
- `references/modes/file-janitor-audit.md` — temp sweep, orphan-file, stale review-bundle, non-canonical residue, and scheduled janitor escalation rules.
- `templates/workspace-verification-checklist.md` — final answer / post-operation verification checklist.
- `references/workspace-organization-reference.md` — source lineage for memory/docs/ops workspace taxonomy.
- `references/file-routing-reference.md` — source lineage for file-routing and naming rules.
