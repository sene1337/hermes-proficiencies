# helpful-hermes-human-playbook-sop-creator

This skill helps Hermes create playbooks, SOPs, operating standards, and review protocols for people.

Use it when the final artifact is meant to be read and followed by a human teammate, not just loaded by an agent at runtime.

## Why it exists

Agents often write procedures that only make sense if you saw the original chat. A useful human SOP needs to stand on its own. It should explain who it is for, when to use it, what the prerequisites are, what to do step by step, when to stop or escalate, and how to check that the work was done correctly.

This skill gives Hermes a structure for writing those artifacts clearly.

## What it creates

- Human playbooks.
- SOPs.
- Operating standards.
- Review protocols and scorecards.
- Human-readable checklists and escalation rules.

## When to use a different skill

If the artifact is meant to teach Hermes a runtime behavior or tool workflow, use `methodical-hermes-skill-tool-builder` instead.

If the artifact is being prepared for public release, use `thorough-hermes-skill-publishing` as the publishing gate.
